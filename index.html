Codenominator Function and the Involution Jimm

The codenominator is a function that extends the Fibonacci sequence to the index set of positive rational numbers, \(\mathbf{Q}^+\). Many known Fibonacci identities carry over to the codenominator. One can express Dyer's outer automorphism \(\alpha\) of the extended modular group \(\text{PGL}(2, \mathbf{Z})\) in terms of the codenominator. This automorphism can be viewed as an automorphism group of the trivalent tree. The real \(\alpha\)-covariant modular function Jimm on the real line \(\mathbf{R}\) is defined via the codenominator. Jimm relates the Stern-Brocot tree to the Bird tree.

Jimm induces an involution of the moduli space of rank-2 pseudolattices and is related to the arithmetic of real quadratic irrationals.

Definition of the Codenominator

The codenominator function \(F: \mathbf{Q}^+ \to \mathbf{Z}^+\) is defined by the following system of functional equations:

\( F\left(1 + \frac{1}{x}\right) = F(x), \quad F\left(\frac{1}{1+x}\right) = F(x) + F\left(\frac{1}{x}\right), \quad x \in \mathbf{Q}^+. \)

The initial condition is given by:

\(F(1) = 1.\)

The function \(F(1/x) \equiv F(1 + x)\) is called the conumerator. The name "codenominator" comes from the fact that the usual denominator function \(D: \mathbf{Q}^+ \to \mathbf{Z}^+\) can be defined by similar functional equations:

\( D(1+x) = D(x), \quad D\left(\frac{1}{1+x}\right) = D(x) + D\left(\frac{1}{x}\right). \)

with the initial condition:

\(D(1) = 1.\)

The codenominator takes every positive integral value infinitely often.

Connection with the Fibonacci Sequence

For integer arguments, the codenominator agrees with the standard Fibonacci sequence, satisfying the recurrence:

\( F_{n+1} = F_n + F_{n-1}, \quad F_1 = F_2 = 1. \)

The codenominator extends this sequence to positive rational arguments. Moreover, for every rational \(x \in \mathbf{Q}^+\), the sequence \(G_n := F(x + n)\) is the so-called Gibonacci sequence (also called the generalized Fibonacci sequence). It is defined by:

\( G_0 := F(x), \quad G_1 := F(1+x), \)

and the recurrence:

\( G_{n+2} = G_{n+1} + G_n. \)

Examples

Here are some examples of the codenominator function:

# Example
1 \(F(n) = F_n\)
2 \(F(1/2) = 2\); more generally, \(F(1/n) = F(n+1) = F_{n+1}\).
3 \(F(n + 1/2) = L_n = F_{n+1} + F_{n-1}\)
4 \(F(n + 1/3) = 2F_{n+1} + F_{n-1}\)
5 \(F(n + 1/4) = 3F_{n+1} + F_{n-1}\)

Properties of the Codenominator

  1. Fibonacci Recursion: \( F(x + n) = F_n F(x + 1) + F_{n-1} F(x), \quad n > 1, x \in \mathbf{Q}^+. \)
  2. Symmetry: \( F(1 - x) = F(x), \quad x \in (0, 1) \cap \mathbf{Q}. \)
  3. Continued Fractions: For a rational number \(x\) expressed as a simple continued fraction \([n_0, n_1, ..., n_k]\), the value of \(F(x)\) can be computed recursively: \( F[n_0, n_1, ..., n_k] = F_{n_0}F[n_1, ..., n_k] + F_{n_0 - 1}F[n_1 + 1, ..., n_k]. \)

Involution Jimm

The Jimm function is defined on positive rational arguments via:

\( J(x) := \frac{F(1/x)}{F(x)}, \quad x \in \mathbf{Q}^+. \)

This function is involutive and admits a natural extension to non-zero rationals via \(J(-x) := -1/J(x)\), which is also involutive.
